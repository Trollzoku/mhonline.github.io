Obtaining and using Hunter Sets

At level 10 and 40 you will get Hunter sets as mission rewards. You can also use Silver Zenny to buy directly from the merchant



On equiping the full set of 6, the first of the key skills for the set will activate.



when you select a skill you would like to unlock, parts required and an amount of Zenny to pay will be shown. Select the number of parts you would like to feed the armor and accept to unlock the skill.



Each skill you unlock will give you points to spend on activating skills, the more points you have the more skills you can activate.



Unlocking skills also activates the key set skills. At 40 unlocked skills all key set skills will be activated.
